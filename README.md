# AsyncAPI (asyncapi)
AsyncAPI is a Linux Foundation project that improves the state of event-driven architectures by providing specifications and tooling for defining asynchronous APIs. It enables developers to document, generate code, and manage message-driven APIs across protocols like Kafka, MQTT, WebSocket, and AMQP.
